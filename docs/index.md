# LLM Foundations

Welcome to **LLM Foundations** — a companion micro-site to the LLM Understanding Lesson Plan.

This course is designed to give students a clear, structured understanding of how Large-Language Model (LLM) systems work, from the core model architecture through the clients, providers, and emerging agent-based capabilities.

---

## Course Outline

This is an ordered course. Work through each section in sequence for the best learning experience.

| # | Section | Description | Status |

| 1 | [LLM Understanding](llm-understanding/index.md) | System context diagrams and definitions of the LLM system | ✅ Available |
| 2 | [LLM Clients](llm-clients/index.md) | Products and interfaces that connect to LLM systems | 🔜 Coming Soon |
| 3 | [LLM Models](llm-models/index.md) | The core model families and their capabilities | 🔜 Coming Soon |
